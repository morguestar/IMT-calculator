# Введение
### Наша команда создает инновационный продукт у которого нет аналогов на рынке - калькулятор ИМТ с внедренной таблицей рекордов, которая будет побуждать людей как можно скорее избавляться от лишнего веса.
### Так же планируется ввести таблицу рекордов по самому низкому ИМТ, чтобы анорексики так же стремились поддерживать свой вес в норме, спасаясь от общественного осуждения.

# Примеры использования

+ При нажатии на кнопку начать перед пользователем откроется таблица, в которую нужно будет вписать свой рост и вес, а так же пол и возраст.
+ При выдаче результатов будет представлены средний результат в возрастной категории пользователя.
+ Так же после получения результатов откроется доступ к таблице лидеров и покажется место пользователя в ней.
+ После получения результатов будет дан выбор пользователю - хочет ли он набрать в весе, или наоборот похудеть.
+ В соответсвтии с выбором будут даны рекомендации
+ Следующий расчёт ИМТ будет доступен ровно через 24 часа с момента прошлого расчёта
+ Участники, которые быстрее всех изменили свое место в лидерборде в соответсвтии с рекомендациями могут претендовать на призы, им будет дана форма на его получение
+ В калькуляторе будет реализована система чатов и друзей
